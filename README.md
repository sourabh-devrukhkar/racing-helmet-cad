# 🏎️ Racing Helmet | SolidWorks Surfacing

High-fidelity surface modeling of an aerodynamic racing helmet. This project emphasizes **G2 Curvature Continuity** and organized **Top-Down Design**.



## 🛠️ Design Highlights
* **Advanced Surfacing:** Built using `Boundary Surfaces`, `Lofted Surfaces`, and `Surface-Knit`.
* **Modular Organization:** 10+ Functional folders for easy parametric editing.
* **Top-Down Logic:** 8 distinct solid bodies managed within a single master part.
* **Custom Detailing:** Integrated "SD" branding using `Split-Line` and `Offset-Surface`.

## 📂 Project Structure
* **`/SolidWork_File`**: Master `.SLDPRT` file.
* **`/Images`**: Renders and Feature Tree screenshots.
* **`/blueprints`**: Reference layout sketches.
